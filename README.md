# Garter_Snake_SR
Code run for Biana Qiu's senior research thesis in 2026: The effects of habitat variables on site occupancy of the Common Gartersnake (Thamnophis sirtalis) on a restored prairie/wetland complex. Analysis ran on data gathered by Biana, Geoffrey R. Smith, Wesley O. Smith and many others throughout 2016-2025 in the Granville Schools Land Lab.
